# ArduinoC

### C++ 輸入與輸出

```c++
//cin_s.cpp
#include <iostream>
using namespace std;

int main() {
	int englishScore = 60;
	cout << "請輸入您的英文分數:";
	cin >> englishScore;
	cout << "Hello ! 您的英文分數是: " << englishScore;
	return 0;
}
```
